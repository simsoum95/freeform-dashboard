# Freeform Dashboard Editor

Grab **any** card on your Home Assistant dashboard and move, resize, and restyle
it freely with the mouse or your finger — no YAML, no rigid grid. It works as a
true overlay on your **existing** dashboard, so cards stay live and editable.

- 🖐️ Free drag & resize of any card (content scales with it)
- 🎨 Per-card style: opacity, blur, rounded corners, shadow, rotate, background — plus one-tap presets
- 🧲 Magnetic alignment guides, multi-select align/distribute, z-order
- 🎬 Save multiple layout "scenes" (Day / Night / Guests) per screen size
- ✏️ Double-click a card to open Home Assistant's **real** native editor
- ↶ ↷ Undo / redo, 🔒 lock cards, ↺ rescue stranded cards
- 🌍 English / French / Hebrew, auto-detected; LTR + RTL
- ☁️ Syncs across your devices

After installing via HACS, add it as a dashboard resource and a small edit
button appears on your dashboard. See the README for setup and configuration.
